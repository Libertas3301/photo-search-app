# Search app using unsplash API

A search app where user could find any photo he wants, and find a lot of photos on each request

## Getting Started

### Prerequisites

```
- install node modules
- Launch app
```

### Installing

At first we will install node modules

```
npm i
```

And after that

```
npm run dev
```
